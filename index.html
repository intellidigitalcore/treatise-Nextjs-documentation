overview

Treatise - NextJs Template

Treatise is an eye-catching NextJs theme luxury design for personal bloggers. With enough theme options to give you plenty of control without being overwhelming with beautiful blog layouts, single post layouts, Post adding in via Rich Text editor and optional plugins available to give your theme an extra boost. it’s super simple and easy to setup,fast and a breeze to use. You will get high quality responsive, well crafted blog out of the box to make writers only focuses on writing content. it has great typography to make your fans and followers focus on every word you write.


It’s built with the most modern tools (HTML5, Tailwind, ES6 and NextJS) . it’s a very complete and highly customizable template that will help you build your own website layouts. It’s Simple, elegant, extensive, professional..

requirements

There are system requirements in order to install and setup the "Treatise" and its components properly. Make sure that you are running the node.js install npm install. You can download Node.js from here: Node JS. NPM comes bundled with Node.js

Recommended node and npm

node install

npm install

Install Npm Packages

NPM : You must have to npm install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.

npm install

Development server

Run npm run start for a dev server. Navigate to http://localhost:3000/ . The app will automatically reload if you change any of the source files

npm run start

Build

Run npm run build to build the project. The build artifacts will be stored in the build/ directory. Use the npm run build --prod flag for a production build.

npm run build

npm run build --prod

More Help

To get more help on the react use npm --help or go check out the NextJs Support.

npm --help

Change Site Title and Favicon

Go to /index.html file to change the title and fav icon

Change Logo

Go to /components/shared-component/header/Header.js file and change logo.

Change Logo

Go to /components/shared-component/loader/Loader.js file and change logo.

Change Basename and Route

You can change the route pages folder. Depending of the file your route will set automatically.

Customize Menu

Go to /components/shared-component/header/Header.js file and change menu items.

Customize Footer

Go to /components/shared-component/footer/Footer.js file and change footer logo, social media, footer data.

Change Copyright Infomartaion

Go to /components/shared-component/footer/Footer.js file and change copyright information.

Section Components

Go to /src/components folder. In components folder you will get all the section components. Go through each components and change section data.

All Blog

Go to /pages/allBlog.js file. Here you will find all the all blogs things.

Single Blog components

Go to /components/single-blog folder. Here you will find all the single-blog components.

Create Blog Page

Go to /pages/create-blog.js file. Here you will find every thing regarding create blogs.

Author Profile Components

Go to /components/author-profile folder to find all the author-profile related components.

MemberShip Component

Go to /components/membership folder to changed anything regarding membership.

Typography - Font Family

Go to /styles/globals.css file and put your font family link to change the font of the website.

Typography - Color

Go to /tailwind.config.js file and put your color code to change font and other colors.
